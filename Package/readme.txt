== PREREQUISITES ==
Cinema 4D R18 or newer.

== INSTALLATION ==
Delete any existing Cinema 4D Source Tool Installations.

Copy the 'Cinema 4D Source Tools' folder into the Cinema 4D Plugins folder.

	C:\Program Files\MAXON\Cinema 4D\plugins
		OR
	C:\Users\$USERNAME$\AppData\Roaming\MAXON\Cinema 4D\plugins
	
Start Cinema 4D and see if the plugin has registered by looking for its
	entry in plugins > Cinema 4D Source Tools.
